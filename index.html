About Me

Manu

Full Stack Developer & Pokémon Enthusiast

Hello! I'm Manu, the creator of this Pokémon Database. As a passionate web developer and lifelong Pokémon fan, I built this project to showcase my skills in front-end development while creating something useful for fellow Pokémon enthusiasts.

This application was built using HTML, CSS, and JavaScript, fetching data directly from the PokéAPI. I've focused on creating a responsive, user-friendly interface that allows you to browse Pokémon by type and explore their stats and evolution chains.
